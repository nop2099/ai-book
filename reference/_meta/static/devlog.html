March 9, 2026
Cribbage and Bridge in one afternoon

Built two complete board games to prove the flywheel. Cribbage: 2.5 hours wall clock, ~15 minutes human attention. Full peg board, scoring engine, AI opponent, training mode with expected value analysis. Bridge: started 40 minutes after cribbage, done same session. Second game was faster because the shape was already learned.

The 15-minute ratio: domain expertise matters more than programming skill. Two games in one afternoon = flywheel proof.
