Shapes of Intelligence

A folder, some files, and an agent that can touch them. That's the whole thing.

The idea: ChatGPT in a browser is a head in a box — smart, but no hands. An agent with file access is an octopus — it can read, write, search, build, and fix. The gap between those two is surprisingly small: a directory, a few text files, and an LLM with permission to act. Everything on this site — the guides, the book, the tools — is built on that. How to use this: Read a page yourself, or paste it into your AI agent: "Read this page and help me get set up." Each page works both ways — human-readable and agent-parseable. This site was built with the tools it describes. See the dev log for how. Trust model: Pasting a URL into an agent is like curl | bash — you're trusting this site to give your agent safe instructions. The source is auditable at GitHub. Content hashes are at hashes.json. Read the page yourself before handing it to an agent if you're not sure.
